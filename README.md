Fitur Utama Aplikasi
CRUD Data Tugas: Menambahkan, melihat, mengedit, dan menghapus tugas.
Status Tugas: Menandai tugas sebagai selesai atau belum selesai.
Filter Tugas: Menampilkan tugas berdasarkan status (Semua, Selesai, atau Belum Selesai).
Dashboard Ringkasan: Menampilkan jumlah total tugas, tugas yang selesai, dan tugas yang belum selesai secara real-time.
Penyimpanan Database: Data tugas disimpan menggunakan database sehingga tetap tersimpan meskipun aplikasi ditutup.
Struktur Folder
frontend/ : Berisi file HTML, CSS, dan JavaScript (client-side).
backend/ : Berisi server Express.js dan API.
database/ : Berisi konfigurasi database SQLite serta file todo.db.
Cara Menjalankan Proyek di Localhost
Pastikan Node.js telah terinstal di perangkat Anda.
Clone atau ekstrak folder proyek, kemudian buka folder utama melalui Terminal atau Visual Studio Code.

Instal seluruh dependency dengan menjalankan:

npm install

Jalankan server backend:

npm start

Buka browser, lalu akses aplikasi melalui:

http://localhost:3000

atau sesuaikan dengan port yang digunakan pada server Anda.
